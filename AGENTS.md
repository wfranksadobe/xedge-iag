# IAG Corporate Website Migration
## Adobe Experience Manager Cloud Service + Edge Delivery Services + Universal Editor

This project is the migration of the Insurance Australia Group (IAG) corporate digital estate located at https://www.iag.com.au to Adobe Experience Manager Cloud Service using Edge Delivery Services (EDS) and Universal Editor (UE).

## Project Overview

Modernise the IAG corporate web platform by migrating legacy website content and authoring workflows into Adobe Experience Manager Cloud Service using:
- Edge Delivery Services (EDS)
- Universal Editor (UE)
- AEM Assets
- Enterprise-grade governance and publishing controls
